### - Major Function
1) 사업자등록증 촬영을 통한 쉬운 회원가입
2) Website의 공고 크롤링 및 앱에서 보여주기
3) 맞춤공고(키워드) Push Alarm

### - My role
1) Google Cloud Vison API을 활용한 이미지 텍스트 추출로 회원가입 내용 쉽게 작성하기
2) Firebase Messaging을 활용한 맞춤공고 Push Alarm
